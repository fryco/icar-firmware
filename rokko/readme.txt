1, Start project: rokko
rokko，是"六甲"的英文翻译，六甲（甲子、甲戌、甲申、甲午、甲辰、甲寅）是阳神，为天帝所役使，能行风雷，制鬼神。在西游记中，与六丁等诸神暗中保护唐憎去西天取经。
取此名寓意暗中保护主人，平安到达目的地。

2, 项目包括在服务器后台运行的 daemon, 在单片机端运行的 client 程序
daemon 运行在 Linux 系统上，目前是在Linux 2.6.38-8-generic #42-Ubuntu SMP 上开发
test 目录是一些测试程序，如验证daemon最大连接数等
client 运行在STM32F103系列单片机上，目前是通过GPRS 连接到daemon 上
